一、可执行文件说明

其中包括3个文件

airport.sql--数据库建表和插入数据的SQL语言。

Server.jar--运行在JAVA(TM)platform SE binary上的socket服务器程序。

Tech.Airport.apk--android客户端程序。

二、运行环境说明

1.客户端为android4.0版本或以上，演示视频和测试为android4.4.4版本。

2.服务器Server.jar需要安装在有Mysql5.0及以上版本的PC上。

  其中：MySQL的用户名应改为“root”；

        MySQL的密码应改为“root”；

        PC防火墙处于关闭状态。

3.确保服务器和客户端在同一个局域网的环境下。

4.运行客户端首页时要输入ip地址，这个地址就是pc在当前局域网中的ip地址。

三、运行源码说明

1.在Eclipse中导入源码时可能程序出现红色感叹号，这是因为程序和slidingmenu的调用关系导致，此时您需要将eclipse-sample 中的slidingmenu单独导入Eclipse一次，同时在Android->Library里导入slidingmenu。

2.由于android环境和sdk版本比较复杂且易出现问题，若在导入过程中出现不能解决的错误，请及时联系开发人员。
